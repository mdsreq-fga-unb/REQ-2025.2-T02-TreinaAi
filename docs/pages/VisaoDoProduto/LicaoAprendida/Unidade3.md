# Unidade 03

Durante o desenvolvimento da terceira fase do projeto TreinaAi, a equipe entrou na etapa mais intensa de implementação, focando na codificação das principais funcionalidades do sistema. Com a base conceitual e estrutural já estabelecida nas fases anteriores, esta unidade foi marcada por avanços significativos, mas também por desafios técnicos que exigiram colaboração, organização e adaptação contínua.

### Aprendizado e Melhorias para o Processo

#### Problemas no ambiente de desenvolvimento

- **Desafio:** Tentativa inicial de usar o Android Studio.  
- **Ação de Melhoria:** Migração para o Visual Studio Code, aumentando a produtividade.

#### Dificuldades na criação de gráficos para visualização de dados

- **Desafio:** Criação dos gráficos necessários para representar os dados das planilhas.
- **Ação de Melhoria:** Adoção do pacote FI Charts, que simplificou a geração dos gráficos e acelerou o processo de implementação.

#### Gestão do tempo

- **Desafio:** Dificuldade em conciliar o tempo do projeto com outras disciplinas e compromissos.  
- **Ação de Melhoria:** Criação de uma agenda semanal para distribuir melhor as atividade
